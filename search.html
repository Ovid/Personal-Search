This is an experiment with HTML5 and JavaScript. If you can't/won't support either, it won't be very interesting to you.

If the first character you type in the search box is a bang (!), then an autocomplete dropdown with different searches wil show up as you start typing. Hiting tab will complete the search type and then you just type the rest of your search and hit enter.

Try these searches:

Description Search
CPAN !cpan aliased
Flickr CC/Commercial !flickrcc paris
Google search !google paris
Google news !news paris
MetaCPAN !metacpan aliased
Personal blog search !publius economics
Wolfram Alpha !wolframalpha what is the sound of one hand clapping
Youtube !youtube michael franti hole in the bucket

Searches not matching desired prefix fall back to duckduckgo.com
